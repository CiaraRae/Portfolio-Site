Project Overview


Ceramic Coffee Mill - Skerton PRO

When approaching my learning for this assignment, I must admit I felt a little lost at first. There were so many helpful resources that were provided, from videos to step-by-step walk-throughs, but since the construction of these learning documents was so new to me I wasn't sure where to begin. Unlike in previous modules, I didn’t have anyone in my personal learning network who was familiar with DITA. Not having peers to go to for help certainly impacted my comprehension of the material at first, so I relied more heavily on resources that were highly visual or descriptive, like code examples and the OXygen DITA Authoring website for quick semantic questions. In fact, I found the DITA sample code to be one of the most helpful tools for checking how tags translated into a final PDF. Once I got the main pages arranged, I started to configure the HTML and PDF transformations so that I could see how my content was performing. Through this process, I learned quickly that while some decisions translate well to a web document, they don’t always do so in the PDF, and visa versa. While I wasn’t able to make every change I wanted to (either because of DITA limitations or my own knowledge limitations), sorting through the pages in each version allowed me to incorporate changes that ensured that both versions were better formatted for readability.
